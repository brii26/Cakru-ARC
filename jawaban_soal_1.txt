merge conflict biasanya terjadi jika terdapat perbedaan code pada line tertentu antara feature branch dan main branch saat merge di eksekusi. hal ini dapat di atasi dengan menentukan line code yang akan menjadi pilihan kita sebagai result commit terbaru. biasanya pada IDE , saat melakukan merge akan di highlight perbedaan code pada line yang sama antara kedua branch sebelum commit final.
