git rebase merupakan kondisi dimana kita mau menggabungkan perubahan / penambahan code pada main branch dari misalkan feature branch. beda dari merge, rebase merupakan penggabungan dengan cara mengubah posisi feature branch pada posisi awalnya di main branch , ke posisi commit trakhir pada main branch, menghasilkan history version commit yang lebih linear, biasanya cara ini dapat mencegah conflict.
