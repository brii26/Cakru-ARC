Axios adalah library javascript yang banyak digunakan untuk membuat HTTP request dari web browser atau applikasi Node.js. library tersebut, menawarkan API sederhana yang mendukung async/await syntax, membuat program menjadi mudah untuk melakukan operasi asynchronous seperti fetching data dari APIs. Dengan fitur seperti interceptors untuk request dan response handling, Axios memiliki flexibilitas  dan custom options. Axios dikenal sebagai library yang cukup mudah untuk digunakan, serba guna, dan popularitas pada komunitas javascript karena dapat menghandle HTTP request secara efisien.
